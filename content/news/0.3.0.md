Parsanol 0.3.0 introduces three powerful new features for extracting and managing parsed data: **Capture Atoms**, **Scope Atoms**, and **Dynamic Atoms**.

## Capture Atoms

Extract named values from your input, like named groups in regular expressions but integrated into the parsing grammar.

```rust
let grammar = GrammarBuilder::new()
    .rule("email", capture("email",
        sequence([
            capture("local", re(r"[a-zA-Z0-9._%+-]+")),
            str("@"),
            capture("domain", re(r"[a-zA-Z0-9.-]+\.[a-zA-Z]{2,}")),
        ])
    ))
    .build();

let result = parser.parse("user@example.com")?;
println!("Local: {:?}", result.get_capture("local", input));
println!("Domain: {:?}", result.get_capture("domain", input));
```

## Scope Atoms

Create isolated capture contexts for nested parsing. Captures made inside a scope are discarded when the scope exits.

```rust
let grammar = GrammarBuilder::new()
    .rule("outer", sequence([
        capture("prefix", str("start")),
        scope(sequence([
            capture("temp", str("inner")),  // Discarded after scope
            str(" "),
        ])),
        capture("suffix", str("end")),
    ]))
    .build();

// Result only contains "prefix" and "suffix"
```

## Dynamic Atoms

Runtime-determined parsing via callbacks. Enable context-sensitive parsing.

```rust
struct TypeCallback;
impl DynamicCallback for TypeCallback {
    fn resolve(&self, ctx: &DynamicContext) -> Option<Atom> {
        match ctx.get_capture_text("type")? {
            "int" => Some(Atom::Re { pattern: r"\d+".into() }),
            "str" => Some(Atom::Re { pattern: r#"[^"]*"#.into() }),
            _ => None,
        }
    }
}
```

## What's Changed

- New `Atom::Capture`, `Atom::Scope`, `Atom::Dynamic` variants
- New `ParseResult` methods: `capture_names()`, `get_capture()`, `captures()`
- `CaptureState` for zero-copy string storage
- Thread-safe dynamic callback registry
- Streaming parser capture persistence across chunks
- DSL helpers: `capture()`, `scope()`, `dynamic_with_id()`
- 400+ tests (up from 396)
